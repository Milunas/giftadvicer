# Gift Advicer

### About
Mobile app for Android to communicate between friends and family what gift for who is the best option to buy.

### Technologies

#### Back-end
- Spring (boot, data, test, web)
- PostgreSQL (for storage data)

#### Mobile
- Kotlin
- Android
- Dagger 2
- RxJava 2 
- Retrofit
- MVP architecture
- Data Binding
